# Keeper App 📝

Keeper App is a simple note-taking web application inspired by Google Keep.  
It allows users to create, view, and delete notes with a clean and intuitive user interface.

## Live Demo
https://keeper-app.vercel.app/

## Overview
This project is a React-based application that demonstrates core frontend concepts such as component-based architecture, state management, and dynamic rendering.

Users can add notes with a title and content, and manage them efficiently in a visually organized layout.

## Features
- Create notes with title and content  
- Display notes dynamically  
- Delete notes with a single click  
- Responsive and minimal UI design  
- Real-time UI updates using React state  

## Tech Stack
- HTML5  
- CSS3  
- JavaScript  
- React.js  

## Key Concepts Covered
- React components and props  
- useState hook for state management  
- Event handling in React  
- Dynamic rendering using `.map()`  
- Component reusability  

## How It Works
The application uses React state to store notes. Each note is rendered dynamically using array mapping, and user actions such as adding or deleting notes update the UI in real time.

## Limitations
- No data persistence (notes are lost on refresh)  
- No backend or database integration  
- No authentication system  

## Future Improvements
- Add LocalStorage for data persistence  
- Integrate backend (Node.js / Firebase)  
- Add user authentication  
- Implement note editing functionality  
- Add categories or labels  

## Deployment
The project is deployed using Vercel.
